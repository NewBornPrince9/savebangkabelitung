# savebangkabelitung
Kepulauan Bangka Belitung (Babel) adalah provinsi di Indonesia yang terdiri dari Pulau Bangka dan Belitung serta pulau-pulau kecil lainnya, terkenal dengan pantai indah, bebatuan granit unik, dan sejarah pertambangan timah serta kaolin, menjadi lokasi inspirasi novel Laskar Pelangi, serta memiliki budaya beragam dari etnis Melayu, Tionghoa.
